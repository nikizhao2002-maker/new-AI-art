# 舞鱼灯 v5 交接文档(2026-06-12)

**启动**:本目录下 `python serve.py`(默认 8081,可 `python serve.py 8084` 指定端口)→ Chrome 开 `http://localhost:<端口>/`。
**调试**:`D` 开发面板 / 数字键 `1-5` 跳阶段 / 空格推进 / `?look=off` 关闭全部视觉调色(回到 v3 原始观感排查问题用)。

---

## 一、相较 v3 的改进总览(给组员同步用)

### 视觉氛围(「水墨入卷 → 灯会之夜」)
- **PHASE_LOOK 调参表**(app.js 顶部):每阶段的曝光/雾/三灯/水墨强度/bloom 全在这一张表,阶段切换平滑过渡。前段水墨写意(青灰宣纸+暖金点睛),末段节庆灯会(暖金朱红)
- ACES 电影感色调映射 + IBL 环境光照(RoomEnvironment)
- **Selective Bloom**(BLOOM_LAYER=11):只有星点/烟花/光晕/孔明灯溢光,鱼灯本体锐利不发雾
- 水彩水面着色器加水墨调色(uInkStrength 等);默认水底色从蓝改青绿(根除蓝底穿帮)
- 夜景:暖星空、12 盏孔明灯上飘、灯群碎金光带、灯光晕、放飞瞬间金色烟花+曝光脉冲
- 加载页/知识卡/HUD 全套水墨化(宣纸纹/朱红印章/双态配色);墨晕转场(只在结尾回开头)

### 游动系统(核心重写)
- **运行时程序化绑骨**(fish-swim.js `buildSpineRig`):静态 GLB 加载时转 SkinnedMesh + 7 节脊柱骨链,行波沿骨骼传播(头稳尾甩);克隆用 `SkeletonUtils.clone + attachSpineRig`(切勿对克隆二次绑骨——共享 geometry 蒙皮会互相污染)
- **三种游动风格**:鱼=行波甩尾 / 虾=收腹弹尾脉冲 / 蟹=横行摇摆(按鱼种名自动分配)
- **鳍部颤动**:着色器按几何位置识别胸/腹/背鳍顶点做高频扇动,与骨骼叠加
- 身体长轴自动检测(这批 GLB 多沿 X 轴!)+ 摆幅按体长归一化

### 交互与流程
- **手势进度环**:金色圆环实时显示识别进度(捏合🤏/OK👌/挥掌🖐/握拳✊→张开✋/✌️)
- **✌️ 手势**:粒子页顺序切换 5 种溶解特效;与 OK 双向互斥防误判
- **倒计时全面可视化**:工艺页大号金字倒计时徽章(手势后看完视频才进下一步+末步 5 秒完成);各推进点要么观众交互、要么明确倒计时,无静默等待
- 所有"自动跳过"按需移除:知识卡片只认 OK;化光页只认 OK;巡游尾声比 OK 回水面(45s 无人值守兜底)
- 卡片展示:鱼在左侧大范围空间游动(双频路线不重复),不再原地旋转;预览鱼缩小不截断

### 工艺环节(outputs-video,两轮重构)
- **实体模型渐进 + 手势揭示制**:进入无模型(微尘)→ 捏合凝出描线稿 → 剪刀手出竹篾骨架(暖边光)→ 握拳张开覆纱 → OK 上色完成;72k 粒子只做工序间 2.5 秒溶解过渡
- 每步手势成功后**看完工艺影像**(徽章倒计时)才揭示下一工序;末步 5 秒倒计时进领灯
- 工序 GLB 在卡片阅读期间后台预热,跳转不卡

### 鱼灯巡游(末段)
- 队形巡游:十盏灯沿河道路线鱼贯而行,20 秒一换「一字长队 ⇄ 环形灯阵」,机头对准航向+入弯滚转
- 灯影流彩:色相沿灯身流动(金调内 ±11.5°,各灯相位错开)
- 个人鱼灯:点亮仪式(凝灯后 2.5 秒烛光渐亮)、光晕随缩放同步、放飞后掌控群游
- 夜景锁定相机旋转(星空不再跟鼠标转),领灯展示改鱼灯自转

### 修掉的 v3 顽疾
- 蓝色背景穿帮(三条路径:returnToReveal/换鱼/默认底色)
- 卡片鱼闪烁(鳞光残值+接触光斑快闪)、鱼灯过曝/发雾(透明渲染残留+雾压主角+bloom 全屏阈值)
- UnrealBloom 破坏画布透明度导致背景变黑(主场景与 ParticleScene 各一次,背景改 WebGL 内绘制)
- 跳阶段工艺 iframe 不关闭、粒子页 UI 重叠、每帧 traverse 5 万面模型取材质(已缓存化)
- 点亮仪式从未触发(lanternGlowing 死代码)、加载文案"五条鱼影"

## 二、关键文件与调参位置

| 要调什么 | 去哪里 |
|---|---|
| 各阶段颜色/曝光/雾/bloom | `app.js` 顶部 `PHASE_LOOK` 表 |
| 鱼游动幅度/频率 | 各加载点 `buildSpineRig/attachSpineRig` 的 opts;全局标定在 `fish-swim.js` `_makeSpineController`(3.1 系数) |
| 鳍颤动幅度 | `fish-swim.js` `applyFinFlutter`(0.22 系数、5.6 频率) |
| 灯影流彩浓淡 | `app.js` `cloneLanternForDance` 里 `uFlowStrength`(0.2) |
| 队形巡游路线/节奏 | `app.js` `formationState` + `updateLanternDanceAnimation` 的 Lissajous 参数(20s 切换) |
| 工艺页步骤/文案/手势/视频 | `outputs-video/index.html` `steps` 数组;倒计时徽章 `#countdown-badge` |
| 粒子页参数 | 左上参数板(运行时);默认值在 `activateParticleScene` 的 opts |
| 手势阈值 | `app.js` `isOkConfirmHand/isVSignHand/updateLanternDanceGesture` 等帧数阈值 |

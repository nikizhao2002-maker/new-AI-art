# 舞鱼灯 v2 交接说明

## 当前版本

本包对应当前可运行版本：

```text
http://localhost:8091/?v=20260609-craft10-ok-release
```

推荐在项目根目录启动本地服务：

```text
python -m http.server 8091
```

然后打开上面的地址。

注意：页面使用 Three.js、MediaPipe、字体等 CDN 资源，演示电脑需要能访问网络，或者提前把依赖本地化。

## 当前流程

```text
水面引鱼
→ 钓鱼
→ 水面 3D 鱼 + 卡片
→ OK 手势或 5 秒后进入工艺
→ 工艺 4 步，每步识别手势后展示约 10 秒粒子
→ 第 4 步结束后自动进入领取鱼灯粒子交互
→ 领取鱼灯粒子交互约 15 秒
→ 聚合为完整 3D 鱼灯
→ OK 手势确认领取
→ 鱼灯舞动
→ 握拳后张开手掌放飞鱼灯
```

## 模型资产调用总览

### 1. 前面鱼池游动 / 钓起鱼影 / 卡片展示

调用文件：

```text
glb-pointcloud.js
FISH_TYPES
```

调用模型文件夹：

```text
assets/models/
```

这批是前面钓鱼环节使用的亮色低/中模资产池，按之前整理逻辑视为 2026-05-31 资产组；后面新增的金鲤、海虾、螃蟹也放在这个同一池里统一调用。

映射关系：

| 鱼种 | 鱼池 / 钓起鱼影模型 | 低模鱼灯备用模型 |
|---|---|---|
| 浮金鱼 | `assets/models/浮金鱼影.glb` | `assets/models/浮金鱼灯.glb` |
| 狮子头 | `assets/models/狮子头鱼之影.glb` | `assets/models/狮子头鱼灯.glb` |
| 石斑鱼 | `assets/models/石斑之影.glb` | `assets/models/石斑鱼灯.glb` |
| 红鲤鱼 | `assets/models/红鲤之影.glb` | `assets/models/红鲤鱼灯.glb` |
| 鲮鱼 | `assets/models/鲮鱼之影.glb` | `assets/models/鲮鱼鱼灯.glb` |
| 金鲤 | `assets/models/金鲤之影.glb` | `assets/models/金鲤鱼灯.glb` |
| 海虾 | `assets/models/海虾之影.glb` | `assets/models/赤焰虾灯.glb` |
| 螃蟹 | `assets/models/螃蟹之影.glb` | `assets/models/金甲蟹灯.glb` |

说明：

- 鱼池游动、钓起水面 3D 鱼、右侧卡片预览都从 `FISH_TYPES` 的 `fish` 字段取模型。
- 卡片名称和文案在 `app.js` 的 `FISH_KNOWLEDGE` 中，顺序与 `FISH_TYPES` 一一对应。
- 这里不调用 50000 面高模，避免前面鱼池阶段过重。

### 2. 工艺四步粒子环节

调用页面：

```text
outputs-video/index.html?v=20260609-craft10-ok-release
```

调用模型文件夹：

```text
outputs-video/models/
```

这批是工艺演示专用模型，随 `outputs-video` 工艺页面一起使用，当前项目内文件日期属于 2026-06-04 工艺页面资产。

| 工艺步骤 | 手势 | 粒子模型 |
|---|---|---|
| 画模描线 | 捏合 PINCH | `outputs-video/models/fish-outline.glb` |
| 熏竹扎架 | 剪刀手 | `outputs-video/models/fish-skeleton.glb` |
| 扪纱描鳞 | 握拳 → 张开 | `outputs-video/models/fish-fabric.glb` |
| 着色点睛 | OK 手势 | `outputs-video/models/fish-colored.glb` |

时间控制：

```text
outputs-video/index.html
CRAFT_REVEAL_HOLD_MS = 10000
```

粒子精细度：

```text
outputs-video/index.html
MAX_POINTS = 72000
```

说明：

- 工艺粒子使用 `MeshSurfaceSampler` 从 GLB 模型表面采样。
- 当前默认保留模型色彩，并叠加工艺阶段色彩编织逻辑。
- 第四步结束后不会显示“左右挥手领取鱼灯”UI，会直接通知父页面进入领取鱼灯粒子交互。

### 3. 领取鱼灯粒子交互

调用文件：

```text
app.js
src/fish-manifest.js
src/particle-scene.js
```

调用模型文件夹：

```text
assets/particle-models-50000/
```

这批是 50000 面左右高精度鱼灯模型，用于 2026-06-06 高精度鱼灯资产组。当前版本中，领取粒子交互、领取 3D 鱼灯、鱼群舞动都统一调用这一批。

时间控制：

```text
app.js
setTimeout(completeLanternParticleReceive, 15000)
```

映射关系：

| 前面钓到的鱼影 | 领取粒子 / 高模鱼灯 |
|---|---|
| 浮金之影 | `assets/particle-models-50000/浮金鱼灯50000.glb` |
| 狮子头鱼之影 | `assets/particle-models-50000/狮子头鱼鱼灯50000.glb` |
| 石斑之影 | `assets/particle-models-50000/石斑鱼灯50000.glb` |
| 火鲤之影 | `assets/particle-models-50000/火鲤鱼灯50000.glb` |
| 鲮鱼之影 | `assets/particle-models-50000/鲮鱼鱼灯50000.glb` |
| 金鲤之影 | `assets/particle-models-50000/金鲤鱼灯50000.glb` |
| 海虾之影 | `assets/particle-models-50000/赤焰虾灯50000.glb` |
| 螃蟹之影 | `assets/particle-models-50000/金甲蟹灯50000.glb` |

说明：

- `src/fish-manifest.js` 中的 `HIGH_DETAIL_LANTERN_MODELS` 按 `FISH_TYPES` 顺序一一对应。
- `CRAFT_INDEX_TO_PARTICLE = HIGH_DETAIL_LANTERN_MODELS`，因此领取粒子交互不会再调用旧的 6.3 暗色模型。
- 领取页背景是岭南水乡夜景。

### 4. 领取鱼灯 3D 展示

调用文件：

```text
app.js
loadLanternMesh()
loadHighDetailLanternTemplate()
```

调用模型文件夹：

```text
assets/particle-models-50000/
```

说明：

- 领取粒子 15 秒结束后，显示同一个鱼种对应的 50000 面高模鱼灯。
- 页面会显示当前获得的鱼灯名称，例如“金鲤鱼灯”“海虾灯”“螃蟹灯”。
- 确认领取使用 OK 手势，鼠标点击按钮是备用入口。

手势实现位置：

```text
app.js
isOkConfirmHand()
updateOkConfirmGesture()
```

### 5. 鱼灯舞动 / 放飞鱼灯

调用文件：

```text
app.js
spawnLanternDanceGroup()
enterLanternDance()
releaseUserLantern()
```

调用模型文件夹：

```text
assets/particle-models-50000/
```

说明：

- 鱼群舞动同样使用 50000 面高精度鱼灯模型。
- 舞动鱼群会按 `FISH_TYPES` 顺序加载所有高模鱼灯模板，并复用模板 clone 出舞动实例。
- 用户自己的鱼灯仍然与前面钓到的鱼影保持一致。

放飞手势：

```text
握拳 → 张开手掌
```

实现位置：

```text
app.js
updateLanternDanceGesture()
releaseUserLantern()
```

## 旧模型说明

项目中仍保留了一些历史资源，例如：

```text
assets/particle-models/
```

当前版本主流程不再使用这批作为领取粒子/鱼灯展示来源；当前主流程的高精度鱼灯统一走：

```text
assets/particle-models-50000/
```

这样可以避免新增鱼种 fallback 到旧的“浮金之影”或旧暗色模型。

## 给组员检查的重点

1. 打开 `http://localhost:8091/?v=20260609-craft10-ok-release`。
2. 钓鱼后确认是否显示水面 3D 鱼 + 卡片。
3. 工艺四步是否每步手势后展示约 10 秒粒子。
4. 第四步结束后是否自动进入岭南水乡领取鱼灯粒子交互。
5. 领取粒子是否约 15 秒后变成完整 3D 鱼灯。
6. OK 手势是否能确认领取。
7. 鱼灯舞动阶段是否通过“握拳 → 张开手掌”放飞。
8. 钓到金鲤、海虾、螃蟹时，后面领取和鱼群是否对应各自高模鱼灯。
